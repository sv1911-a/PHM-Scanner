# Code of Conduct

SPECTRE should be a welcoming project for students, researchers, defenders, CTF players, and security professionals.

## Expected behavior

- Be respectful.
- Explain technical feedback clearly.
- Assume good intent.
- Help beginners learn.
- Keep discussions focused on improving the project.

## Unacceptable behavior

- Harassment or personal attacks.
- Sharing private data without permission.
- Encouraging unauthorized activity.
- Submitting features intended primarily for abuse.

## Enforcement

Maintainers may close issues, reject pull requests, or block users who violate this code of conduct.
